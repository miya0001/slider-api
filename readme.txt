This plugin delivers the articles in custom post types as json api for slider images.

== Description ==

* This plugin delivers the articles in custom post types as json api for slider images.
* When developing an image slider, you can concentrate on javascript only.
* To add a custom post type, make more api endpoints.

= Support =

* https://github.com/miya0001/slider-api

== Installation ==

* A plug-in installation screen is displayed on the WordPress admin panel.
* It installs it in `wp-content/plugins`.
* The plug-in is made effective.
